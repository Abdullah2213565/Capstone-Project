## Getting Started

To get a local copy up and running follow these simple example steps.

`gh repo clone Abdullah2213565/Capstone-Project`

### Setup

Dowload NPM and NODE
### Install

Install the packages globally and after that:

`npm install`

### Run tests

`npx hint .`
`npx stylelint "**/*.scss"`
`npx eslint .`
